# How To Record Your Screen On Windows

## Best Option: Snipping Tool Screen Recording

Use this if your Windows computer has the newer Snipping Tool.

### Steps

1. Click the Windows Start button.
2. Search for `Snipping Tool`.
3. Open `Snipping Tool`.
4. At the top of Snipping Tool, choose the **Record** option. It usually looks like a video camera icon.
5. Click `New`.
6. Drag to select the full screen, or as much of the screen as possible.
7. Click `Start`.
8. Wait for the countdown.
9. Complete the task from `WORKER_INSTRUCTIONS.md`.
10. When finished, click the red stop button.
11. Save the video file.
12. Name it exactly as instructed, for example:

```text
video-001-clean-customer-update.mp4
```

## If Snipping Tool Does Not Have Record

Use one of these options:

### Option A: Xbox Game Bar

This may work for browser/app windows, but may not record the full desktop or File Explorer on every computer.

1. Open the main app or browser window you want to record.
2. Press:

```text
Windows key + G
```

3. In the Capture panel, click the record button.
4. If asked, confirm that this is a game/app.
5. Complete the task.
6. Stop the recording.
7. Recordings are usually saved in:

```text
Videos/Captures
```

If Xbox Game Bar refuses to record, use Snipping Tool or ask for help.

### Option B: PowerPoint Screen Recording

Use this if Microsoft PowerPoint is installed.

1. Open PowerPoint.
2. Create a blank presentation.
3. Click `Insert`.
4. Click `Screen Recording`.
5. Select the area to record.
6. Click `Record`.
7. Complete the task.
8. Stop the recording.
9. Right-click the recorded video inside PowerPoint.
10. Choose `Save Media As`.
11. Save the video with the correct filename.

## Important Recording Rules

- Record the screen, not your webcam.
- Do not talk during the recording.
- Keep the mouse cursor visible.
- Do not pause the recording during the task.
- Do not edit the recording afterward.
- Make sure the final video opens and plays before sending it back.

## If You Cannot Record

Stop and ask the project manager for help. Do not continue the task without recording.
