# Windows Video Playback Guide

Use this guide when labeling videos on Windows.

## Recommended App

Use the built-in Windows app called:

```text
Media Player
```

It can play `.mp4` videos and shows the playback time.

## Open A Video

1. Find the `.mp4` video file.
2. Right-click the file.
3. Click `Open with`.
4. Click `Media Player`.
5. If `Media Player` is not shown, click `Choose another app`.
6. Select `Media Player`.
7. Click `Always` or `Just once`. Either is fine.

## Read The Timestamp

When the video is playing, move your mouse over the video.

Look at the playback bar at the bottom.

You should see something like:

```text
0:45 / 1:45
```

The first time is the current position.

In this example, the current timestamp is:

```text
45 seconds
```

## Convert Minutes To Seconds

The labeling tool uses seconds.

Use this table:

| Media Player time | Type in labeling tool |
|---|---:|
| `0:05` | `5` |
| `0:10` | `10` |
| `0:30` | `30` |
| `0:45` | `45` |
| `1:00` | `60` |
| `1:10` | `70` |
| `1:30` | `90` |
| `2:00` | `120` |
| `2:15` | `135` |
| `3:00` | `180` |

Formula:

```text
minutes x 60 + seconds
```

Example:

```text
2:15 = 2 x 60 + 15 = 135 seconds
```

## Pause And Rewind

Use the video controls:

- click the play/pause button to stop or start
- drag the progress bar to move forward or backward
- use the keyboard spacebar to play/pause if it works

It is okay to pause often.

## Put Video And Helper Tool Side By Side

This makes labeling easier.

1. Open the video in Media Player.
2. Open `simple-labeling-tool.html` in Chrome or Edge.
3. Click the Media Player window.
4. Hold the `Windows` key and press `Left Arrow`.
5. Click the browser window.
6. Hold the `Windows` key and press `Right Arrow`.

Now the video and helper tool should be side by side.

## If Media Player Does Not Work

Try the Photos app:

1. Right-click the video.
2. Click `Open with`.
3. Click `Photos`.
4. Use the playback bar and timestamp in Photos.

If neither Media Player nor Photos works, tell your manager:

```text
I cannot open the video in Media Player or Photos.
```

Do not install new software unless your manager says to.
