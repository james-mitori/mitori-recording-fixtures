# Video 001 Gold Label Guide

Video file:

```text
video-001-clean-customer-update.mp4
```

First-pass label should be based on the video only.

Look for:

- what source file is opened
- which customer/entity is worked on
- which destination system/page is used
- what fields appear to be changed
- whether a save or completion confirmation is visible
- whether final visible values line up with source values
- any discrepancy or uncertainty

Do not copy the worker script as the answer. The video is the source of truth.
